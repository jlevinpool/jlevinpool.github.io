HTML5/JavaScript Tower Defense Game

This is just the best. Bar none. Easy, simple, but deviously hard. I loved it.
I kept playing long after I should have moved on to the next project.
Benjamin Brewster, Instructor
Oregon State University

Developed as the capstone project for Oregon State University's Bachelor's Degree in Computer Science, this tower defense game was developed using HTML5 and JavaScript. The objective is to prevent the ever increasing waves of enemy units from reaching the end of the path by building and upgrading attack towers.

My contrubition to the project included:

  • Game logic & rendering loops
  • Game state transition design
  • Unit coloring algorithm
  • Unit pathing algorithm
  • Game unit &mp tower class structure and base functionality

This project was the result of collaboration with my fellow students:
  Brandon Gipson & Thomas Dale
